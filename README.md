# Portfolio (statique)

Un portfolio simple et moderne (HTML/CSS/JS) avec les sections :
Accueil, Présentation, Entreprise, Tâches, Projets, Veille techno.

## Ouvrir

Ouvre `index.html` dans ton navigateur (double‑clic).

## Personnaliser rapidement

- Nom, email, zone : dans `index.html`
- Couleurs / style : dans `styles.css`
- Thème clair/sombre + menu mobile : dans `main.js`
